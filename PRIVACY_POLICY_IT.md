# Informativa sulla Privacy di DomoMeter

Ultimo aggiornamento: 2026-03-17

La presente Informativa sulla Privacy descrive come `DomoMeter` gestisce i dati.

## Sintesi

`DomoMeter` è progettata per funzionare localmente sul dispositivo dell'utente.

- Non è richiesto alcun account
- Non vengono utilizzati strumenti di analytics
- Non vengono utilizzati SDK pubblicitari
- Non viene effettuato alcun tracciamento dell'utente
- Lo sviluppatore non raccoglie dati personali tramite l'app

## Dati trattati dall'app

L'app consente all'utente di salvare e gestire localmente informazioni relative ai consumi domestici, tra cui:

- letture dei contatori di luce, gas e acqua
- nomi dei profili abitazione inseriti dall'utente
- preferenze come lingua, tema, stile grafico, limiti, unità di misura e stato dell'app
- file di backup in formato JSON generati dall'utente

Queste informazioni vengono salvate localmente sul dispositivo per consentire il funzionamento dell'app.

## Raccolta dei dati

Lo sviluppatore non raccoglie, non riceve, non trasmette, non vende e non condivide dati dell'utente tramite server remoti attraverso l'app.

Se l'utente sceglie di esportare o condividere un file di backup, tale azione viene avviata manualmente dall'utente tramite le funzionalità di condivisione del sistema operativo. Eventuali trasferimenti successivi dipendono esclusivamente dalla destinazione scelta dall'utente.

## Condivisione dei dati

`DomoMeter` non condivide dati dell'utente con lo sviluppatore o con terze parti per finalità di analytics, pubblicità, profilazione o tracciamento.

L'unica eccezione è il caso in cui l'utente esporti o condivida esplicitamente un file di backup usando gli strumenti di condivisione della piattaforma. In tal caso, il file viene inviato solo verso la destinazione scelta dall'utente.

## Conservazione dei dati

I dati dell'app vengono salvati localmente sul dispositivo usando lo storage applicativo locale.

I file di backup creati dall'utente vengono salvati solo nella posizione scelta dall'utente oppure in una posizione temporanea usata per supportare la funzione di condivisione.

## Conservazione e cancellazione

I dati inseriti nell'app restano sul dispositivo finché l'utente non:

- modifica o elimina manualmente le letture
- importa un backup che sostituisce i dati locali correnti
- esegue il reset del database locale
- disinstalla l'app, nei limiti del comportamento del sistema operativo
- elimina manualmente i file di backup esportati dalle proprie posizioni di archiviazione

Poiché lo sviluppatore non riceve né conserva i dati dell'app su server remoti, non può accedere, correggere o cancellare i dati locali dell'utente per suo conto.

## Minori

`DomoMeter` non è rivolta specificamente ai minori e non raccoglie consapevolmente dati personali di minori.

## Sicurezza

L'app si affida ai meccanismi di sicurezza forniti dal sistema operativo e dallo storage locale dell'app. Tuttavia, nessun metodo di archiviazione su dispositivo o di trasferimento file può essere garantito come completamente sicuro, in particolare dopo che l'utente esporta o condivide un file di backup al di fuori dell'app.

## Modifiche a questa informativa

La presente Informativa sulla Privacy può essere aggiornata nelle future versioni dell'app. L'ultima versione dovrebbe essere resa disponibile all'URL pubblico della privacy policy usato in App Store Connect e Google Play Console.
